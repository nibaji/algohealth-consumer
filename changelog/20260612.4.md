# Release 20260612.4

- 2026-06-12: Installed `expo-dev-client` dependency for Android development client builds. Changed in `package.json` and `bun.lock`.
- 2026-06-12: Standardized `KeyboardAvoidingView` logic/props across all screens/modals containing text inputs using a newly created `useKeyboardAvoiding` hook (configuring `behavior="padding"` and conditional `enabled` based on platform and keyboard visibility). Changed in `hooks/useKeyboardAvoiding.ts`, `hooks/useKeyboardVisibility.ts`, `components/medical-records/consult-modal.tsx`, `components/medical-records/edit-member-modal.tsx`, `app/(auth)/login.tsx`, `app/(auth)/register.tsx`, `app/(auth)/forgot-password.tsx`, `app/family/create.tsx`, `app/family/add-member.tsx`, `app/family/join.tsx`, and `app/medical-records/create.tsx`.
- 2026-06-12: Fixed file upload crashes ("Creating blobs from 'ArrayBuffer' and 'ArrayBufferView' are not supported" on Hermes and "Unsupported FormDataPart implementation" under fetch) by routing FormData requests via XMLHttpRequest in the API client and using standard React Native `{ uri, name, type }` file parts. Changed in `src/services/api/apiClient.ts` and `app/medical-records/create.tsx`.
- 2026-06-12: Replaced static audio file picker with an interactive "Audio Note" recording component. Implemented native microphone permission handling, real-time recording timer with a pulsing record animation, and a custom audio player supporting play/pause, dynamic duration metrics, and seek capabilities via progress bar press gesture. Changed in `package.json`, `app.json`, `components/ui/icon.tsx`, and `app/medical-records/create.tsx`.
- 2026-06-12: Wrapped the profile settings layout in KeyboardAvoidingView using the useKeyboardAvoiding hook to prevent software keyboard overlaps. Implemented picked document deduplication by a combination of filename and size in the medical record creation flow to prevent duplicate uploads. Changed in `app/profile.tsx` and `app/medical-records/create.tsx`.
- 2026-06-12: Implemented automatic family invite acceptance upon login, registration, and session restoration. If a pending invite payload is detected in the authentication response, the client automatically requests to join the family circle, updates local family context state, and transitions directly to the home screen. Changed in `src/features/auth/authTypes.ts`, `src/services/auth/authService.ts`, and `src/contexts/AuthContext.tsx`.
